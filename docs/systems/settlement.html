정착 전 콘텐츠 - 소규모 농장 대여/대리 운영 (CLAUDE.md "핵심 시스템 4" 상세본)

확정된 원칙

  • 정식 정착(및 결혼)은 메인 스토리 완료 후 최종 콘텐츠.
  • 스토리 진행 중에는 마을별 소형 필지(3x3~5x5)를 대여하거나 NPC 농장을 대리 운영하며 특산물 체험.
  • 트리거 예시: 마을 도착 시 "여행 다녀올 동안 농장 좀 봐달라" 류의 자연스러운 퀘스트.
  • 소형 농장에서 얻은 재화/작물은 해당 마을 한정 재화로 국한 → 추후 정착 마을 선택 유도 장치.

제안 — 튜토리얼 홈스테드 (2026-07-07, 회의록, 마을1 검증 전까지 제안 상태)

마을1(이슬터)의 "개간 전 농장(잡초·돌·그루터기를 치우는 것 자체가 초반 콘텐츠)" 컨셉은 위 "소형 필지(3x3~5x5)" 규모로는 볼륨이 안 나온다는 문제 제기(game-designer)에서 나온 절충안. 기존 2단계(대여/대리 소형 필지 → 엔딩 후 정식 정착) 사이에 제3의 카테고리를 둔다:

  • 마을1 한정, 스토리 초반부터 플레이어 소유인 시작 거점 — 다른 마을들의 "임시 대여" 취급과 다름.
  • 정식 정착(엔딩 후, 결혼 등 최종 콘텐츠)만큼의 지위는 아님 — 중간 형태.
  • 정확한 크기, 정식 정착으로의 전환 여부, 세이브 스키마상 다른 마을 소형 필지와의 구분 방식은 미정 — 마을1 버티컬 슬라이스 제작 시 systems-designer가 구체화.

미확정 — 상의 필요

  • 마을 한정 재화를 정착 후에도 계속 쓸 수 있는지, 정착 안 한 마을의 한정 재화/작물은 어떻게 처리되는지 (사장 자원화? 최종 정산 보상?)
  • 소형 농장 운영 중 실패(작물 고사 등)에 대한 페널티가 있는지, 있다면 어느 정도로 관대해야 하는지
  • 튜토리얼 홈스테드 구체 규칙(크기, 정식 정착 전환 여부, 세이브 스키마) — 위 "제안" 섹션 참고

다음 작업 후보

  • 소형 농장 규칙(작물 종류 제한, 수확 주기) 수치 설계 (systems-designer)
