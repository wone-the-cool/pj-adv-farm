CLAUDE.md의 "1. 지역/마을 구조" 요약 표를 상세화한 문서. 마을별 시놉시스/캐릭터 시트는 각 마을 하위 섹션이 커지면 docs/villages/<마을명>.md로 추가 분리한다.

마을 후보 (구 초안 — 2026-07-07 신규 방향으로 대체됨, 히스토리 참고용)

마을고유 작물가공품던전 테마결혼 가능 NPC 계열개발 우선순위
사막 오아시스마력 선인장마법 염료고대 유적상인 계열미정
설원 산악서리 열매얼음 조각 장비얼음 동굴대장장이 계열미정
숲속 정령정령 버섯치유 물약태고의 숲정령사 계열미정
해안 어촌해조류(미정)심해 유적(미정)미정

제안된 3마을 컨셉 (2026-07-07, 회의록 — 마을1 버티컬 슬라이스 검증 전까지는 "제안" 상태, 확정 아님)

사용자가 직접 지정한 방향: 최소 3마을, 마을당 평균 플레이타임 ~10시간 목표. 실제 구현은 마을1부터 순차 진행하며, 마을2/3 세부는 마을1 검증 후 확정한다.

마을컨셉고유 자원가공품던전 테마결혼 NPC 계열
1. 이슬터(가칭)튜토리얼, 봄/평지, 개간 전(잡초·돌·그루터기) 튜토리얼 홈스테드새싹밀갓 구운 빵/에일잡초 밑 매몰 유적농부/장인 계열
2. 다래계곡(가칭)열대/여름, 계곡 채집·낚시 (스타듀밸리 진저섬 참고)불꽃과실, 바나나/망고 등 열대 작물과실주/잼정글 신전 + 온천 동굴채집꾼·약초치유사 계열
3. 청파나루(가칭)대서양/여름, 바다낚시·통발·해변 조개/산호 채집 (기존 "해안 어촌" 통합)해조류, 조개, 산호훈제어물/진주세공품심해 유적뱃사람·항해사 계열

확정된 사항

  • 마을 개방 구조 (2026-07-03 확정, 회의록): 아키텍처는 씬 단위 분리 + "잠금 여부"는 세이브에 별도 저장하지 않고 다른 상태값에서 파생되는 계산값으로 처리 (정책 전환 시 마이그레이션 불필요). 정책은 물리적으로 전부 열되 정교한 지형 유도는 만들지 않고 약한 인센티브만 두며, 마을 수가 늘어나는 확장팩 시점에 BOTW식 정교한 유도 설계(지형/랜드마크)에 투자한다.
  • MVP 마을 수/개발 시퀀싱 (2026-07-07 확정, 회의록): 비전은 최소 3마을(위 표)이지만, 실제 구현은 마을1(이슬터) 버티컬 슬라이스부터 — 재밌다고 검증된 후 마을2/3을 확정 진행한다. 해안 어촌 컨셉은 마을3(청파나루)으로 흡수, 가공품/결혼계열 공백 해소됨.

미확정 — 상의 필요

  • 마을2/3 구체 밸런싱 수치(가격/희귀도) — systems-designer, M1 단계에서
  • 마을2/다래계곡·마을3/청파나루 메커닉 차별화(능동 낚시 미니게임 vs 크랩팟 방치형)의 구체 스펙
