문서 지도

CLAUDE.md는 세션마다 자동 로드되므로 철학/원칙/인덱스만 유지하고, 세부 내용은 이 아래 docs/에 분리했다. 2026-07-03부로 docs/는 html이 원본이다 (과거 md 원본은 old/docs/에 보관).

│   ├── villages.html          # 1. 지역/마을 구조

│   ├── romance-companion.html # 5. NPC 로맨스/컴패니언

사용 규칙

  • 세부 내용(마을 시놉시스, 캐릭터 시트, 밸런싱 수치)이 늘어나면 docs/villages/<마을명>, docs/characters/<이름> 등으로 추가 분리한다 (확장자는 .md/.html 중 미확정 — 작성 스킬 사용 시 먼저 확인).
  • 각 문서 상단 메타데이터 표의 status(draft/living-document 등)와 owner_agent로 어떤 에이전트가 주로 다루는 문서인지 표시한다.
  • 미확정 항목은 항상 "미확정 — 상의 필요" 섹션에 옵션 형태로만 기록한다 (CLAUDE.md 작업 원칙과 동일).
  • docs/*.html을 직접 Edit할 때는 태그 구조(<table>, <ul> 등)를 깨뜨리지 않도록 주의한다.

과거 markdown 원본 (old/docs/)

2026-07-03 이전에는 docs/*.md가 원본이고 tools/docs-to-html.js(docs-html-export 스킬)로 docs-html/을 생성하는 구조였다. 지금은 그 산출물이 docs/로 승격되었고, 과거 md 원본은 old/docs/에 히스토리 보관 중이다. docs-html-export 스킬은 old/docs/ 기준 재생성이 필요한 예외 상황에만 쓰는 보류 상태 도구로 남아있다.
