초기 마비노기(G1~G3) 기획 철학 참고자료

넥슨 마비노기 초기작(G1~G3, 엘프/자이언트 출시 시점까지)의 기획 철학을 분석해 이 프로젝트에 적용 가능한 형태로 정리한 사용자 제공 참고자료. "판타지 라이프" 슬로건이 시스템 차원에서 실제로 구현되어 있던 시기(2004~2006년경)의 설계 원칙에 초점.

원작 분석 요약

  • 무클래스 탤런트 + 단일 AP 시스템: 전투 재능과 생활 재능(요리/대장기술/재봉/채집/음악/상행)이 하나의 AP 풀·랭크업 시스템을 공유. 별도 직업 개념 없이 순수 생활형 캐릭터도 전투 캐릭터와 동등한 성장 곡선/존재감을 가짐 — "전투는 삶의 일부일 뿐, 특별한 상위 콘텐츠가 아니다"를 자원 시스템 자체로 구현.
  • 종족-환경-서사의 결합(엘프/자이언트): 신의 저주로 콘누스(엘프, 사막화)/피시스(자이언트, 설원화)가 형성됐다는 세계관이 종족별 스탯 성향, 마을 풍경, 날씨 시스템까지 관통. 환경→생존 문제→종족 서사→캐릭터 정체성이 하나의 인과 사슬.
  • 제네레이션(G1~G3) 구조: 짧고 완결된 챕터 단위 + "무엇이 진정한 정의인가"를 계속 되묻는 도덕적 분기(팔라딘 vs 다크나이트 전향 등). 플레이어의 선택이 실제 정체성(스킬/전향)으로 이어짐.
  • 반응형 액션 전투: 자동사냥 사실상 불가능, 예비동작을 보고 카운터/디펜스로 받아치는 구조.
  • 생활 스킬의 "노동감": 느린 랭크업, 제작 실패 확률 존재 — 편의성보다 "진짜 노동"에 가까운 체감을 의도적으로 유지.
  • 마을/던전 디자인: 마을이 스토리 무대와 밀접 결합, 마을 이동 자체가 세계를 알아가는 여정. 던전은 정해진 경로+서사적 연출(컷신/보스 대사) 중심.

지금 트렌드와의 접점/충돌

  • 잘 맞는 지점: 선택이 시스템에 새겨지는 서사 통합형 설계, 짧고 완결된 챕터 단위 공개.
  • 충돌 지점: 원작 특유의 높은 마찰(반응형 전투 난이도, 노동감, 느린 페이스)은 지금 유저의 마찰 관용도와 안 맞음 — 그대로 이식하면 이탈률 상승 위험. 전투/생활 완전 통합은 세그먼트별 리텐션 관리를 어렵게 함. 초반 "무명·수동적" 주인공 포지션은 파워판타지 기대 유저에게 이탈 지점 가능.
  • 절충 원칙: "통합의 감성은 가져오되, 통합의 강제성은 버린다" — 전투/생활을 상호 보너스로 느슨하게 엮고, 도덕적 분기는 유지하되 결과가 파워 밸런스에 손해를 주지 않도록.

적용 가이드 요약

원작 요소적용 방향
AP 통합(전투=생활)농사↔모험 자원 순환 + 상호 보너스, 강제 결합 지양 — 순수 농사파/순수 모험파 모두 각자 루트로 완주 가능해야 함
종족-환경-저주 서사지역별 작물/기후에 세계관적 인과(저주/전설) 부여
제네레이션 구조짧고 완결된 챕터 + 도덕적 분기(파워 손해 없이)
반응형 전투일상은 자동화, 클라이맥스(보스)만 액션성 유지
날씨·계절 순환별도 트랙이 아닌 세계 순환의 일부로 통합, 서사 비트로 리텐션 견인

핵심 원칙 한 줄 요약: 통합의 감성(전투와 생활이 하나의 삶이라는 감각)은 최대한 살리되, 통합의 강제성(자원 강제 결합, 높은 마찰, 손해 보는 선택)은 현재 유저 기대치에 맞게 완화한다.
