아트 레퍼런스 무드보드 (뼈대 — 실제 이미지 없음)

일반 게임 기획 제안서 표준 목차 신설 항목. 현재 실제 레퍼런스 이미지/링크는 없으며, 항목 뼈대만 마련.

마을별 무드 방향 (텍스트 설명 초안 — 이미지 미첨부)

마을색감/분위기 키워드 (가설)참고할 만한 레퍼런스 장르
사막 오아시스따뜻한 황토/청록 대비, 고대 유적의 신비함사막 오아시스형 픽셀아트 타일셋
설원 산악차가운 청백색, 대장간의 주황 불빛 대비겨울/설산 배경 픽셀아트
숲속 정령짙은 녹색+발광 버섯/정령 이펙트신비로운 숲 판타지 아트
해안 어촌밝은 파랑/모래색, 어촌 특유의 나무 질감해안 마을 픽셀아트

미확정 — 상의 필요

  • 실제 무드보드 이미지(핀터레스트/레퍼런스 게임 스크린샷 등)는 사용자가 직접 수집해 이 문서에 링크/첨부하는 방식 권장 (Claude가 임의로 이미지 URL을 생성하지 않음)
  • 도트 밀도(16x16 vs 32x32), 캐릭터 비율(SD 2~3등신 vs 정상 비율)은 docs/format/game-format-tbd.html와 연동해 결정

다음 작업 후보

  • 사용자가 참고 이미지를 수집하면 이 문서에 정리
  • 확정된 아트 방향을 docs/format/game-format-tbd.html의 그래픽 스타일 항목에 반영
