# 타겟 플랫폼 / 유저층 (뼈대 — 미확정)

일반 게임 기획 제안서 표준 목차 신설 항목. 시장 조사나 팀 의사결정 없이 임의로 확정하지 않는다.

## 결정이 필요한 질문
- **플랫폼**: PC(Steam) 우선? 콘솔(Switch 등) 동시/후속 이식? 모바일 이식 고려 여부? → docs/format/game-format-tbd.md의 엔진/UI 결정과 직결
- **타겟 연령/유저층**: 스타듀밸리/룬팩토리 기존 유저층(캐주얼~중간 난이도 힐링 게임 선호층)을 그대로 겨냥할지, 야숨의 액션/탐험 지향 유저까지 넓힐지
- **세션 길이 설계**: "오늘은 농장만 하고 쉬어도 되는" 철학(CLAUDE.md 4번)과 모험 세션의 길이가 상충하지 않는지 — 1회 플레이 세션 길이 목표치
