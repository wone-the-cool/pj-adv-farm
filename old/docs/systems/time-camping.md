# 시간 시스템 - 리얼타임 + 노숙 (CLAUDE.md "핵심 시스템 3" 상세본)

## 확정된 원칙
- 모험 중에도 시간은 실시간으로 흐름 (마을/필드 간 시간축 분리 없음).
- 스타듀밸리식 오전 2시 강제 기절/귀가 없음 — 대신 **노숙(캠프파이어)**으로 필드에서 밤을 지샐 수 있음.
- 노숙은 선택적 콘텐츠: 자정 전후 특수 몬스터/이벤트, 캠프파이어 동료 상호작용/대사, 야간 전용 채집 아이템 등 보상형 콘텐츠 제공.

## 미확정 — 상의 필요 (CLAUDE.md 리스크 4번)
- 노숙 보상의 구체적 종류/빈도: 매일 밤 보상이 있으면 사실상 필수화될 위험, 너무 드물면 매력이 없음 → 빈도 밸런스 옵션 필요
- 강제 없이도 "하고 싶어지는" 유인 설계: 한정 재화? 스토리 전용 이벤트? 컴패니언 호감도(옵트인 로맨스와 연동 여부)?
- 노숙 중 리스크(몬스터 습격 등)를 얼마나 강하게 걸지 — 너무 약하면 의미 없고, 너무 강하면 페이스를 해침
