# 정착 전 콘텐츠 - 소규모 농장 대여/대리 운영 (CLAUDE.md "핵심 시스템 4" 상세본)

## 확정된 원칙
- 정식 정착(및 결혼)은 메인 스토리 완료 후 최종 콘텐츠.
- 스토리 진행 중에는 마을별 소형 필지(3x3~5x5)를 대여하거나 NPC 농장을 대리 운영하며 특산물 체험.
- 트리거 예시: 마을 도착 시 "여행 다녀올 동안 농장 좀 봐달라" 류의 자연스러운 퀘스트.
- 소형 농장에서 얻은 재화/작물은 해당 마을 한정 재화로 국한 → 추후 정착 마을 선택 유도 장치.

## 미확정 — 상의 필요
- 마을 한정 재화를 정착 후에도 계속 쓸 수 있는지, 정착 안 한 마을의 한정 재화/작물은 어떻게 처리되는지 (사장 자원화? 최종 정산 보상?)
- 소형 농장 운영 중 실패(작물 고사 등)에 대한 페널티가 있는지, 있다면 어느 정도로 관대해야 하는지
