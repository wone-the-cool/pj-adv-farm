# 게임 형식 관련 논의점 (미확정 — 상의 필요)

CLAUDE.md에서 이동. technical-advisor 에이전트가 주로 참고/발전시키는 문서.

- **그래픽 스타일**: 스타듀밸리 유사 2D 픽셀아트 선호. 해상도/도트 밀도(16x16 vs 32x32 등), 캐릭터 비율(SD형 2등신~3등신 vs 정상 비율) 미정.
- **시점/카메라**: 탑다운(스타듀밸리식) 고정, 혹은 쿼터뷰 등 변형 여부 결정 필요.
- **전투 조작 방식**: 스타듀밸리처럼 단순한 실시간 액션(휘두르기, 회피) 기반 선호. 룬팩토리식 스킬/콤보 시스템까지 갈지, 최대한 단순하게 유지할지 결정 필요.
  - 참고: 조작 난이도를 낮게 유지하는 것이 "농장 경영이 메인"이라는 정체성과도 부합함. 전투가 지나치게 복잡해지면 서브 시스템이 메인을 잠식할 위험 있음.
- **엔진/플랫폼**: 미정 (예: 유니티/갓엇, PC 우선 여부, 콘솔/모바일 이식 고려 여부)
- **UI/조작 방식**: 키보드+마우스 기준인지, 게임패드 지원까지 고려하는지
- **세이브/진행 구조**: 야숨식 자유도를 지원하려면 세이브 데이터에 마을별 진행도, 대여 농장 상태, 특산물 획득 여부 등을 어떻게 구조화할지 초기 설계 필요
