# 스토리라인 관련 논의점 (미확정 — 상의 필요)

CLAUDE.md에서 이동. narrative-writer 에이전트가 주로 참고/발전시키는 문서.

- **메인 플롯의 큰 줄기**: 왜 플레이어가 여러 마을을 돌아다니게 되는가에 대한 동기 부여 장치 필요 (예: 몰락한 고향을 재건하기 위한 단서를 찾아 각지를 순회 / 전설의 농장을 물려받았는데 그 힘이 여러 마을에 흩어져 있음 등)
- **마을 간 서사적 연결고리**: 각 마을 에피소드가 완전히 독립적인 옴니버스 구조인지, 아니면 공통된 메인 빌런/사건이 배후에 있고 마을마다 그 사건의 다른 단면을 보여주는 구조인지
- **엔딩 구조**: 정착 마을 선택이 곧 엔딩 분기가 되는지, 혹은 모든 마을을 다 둘러봐도 정착과 무관하게 공통 엔딩으로 수렴하는지
- **보스/빌런 설정**: 각 마을 보스가 개별 위협인지, 더 큰 존재의 하수인/파편인지 (스토리 상 연결성과 직결)
- **회수되지 않은 마을 콘텐츠 처리**: 정착하지 않은 마을들의 서브플롯이나 로맨스 라인을 스토리상 어떻게 자연스럽게 매듭짓거나 열어둘지 ("숨겨진 콘텐츠 낭비" 문제와 연결)
- **텍스트 볼륨**: NPC 상호작용 비중을 낮게 유지하기로 한 만큼, 스토리 전달을 대사량보다 환경 스토리텔링(마을 풍경, 유적, 아이템 설명 등)에 얼마나 의존할지
