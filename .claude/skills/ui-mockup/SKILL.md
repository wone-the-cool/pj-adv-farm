# UI/맵 목업 스킬 (스텁)

**현재 상태: 스텁.** 이 스킬은 아직 완성되지 않았다 — 완성하려면 `docs/format/game-format-tbd.html`의 그래픽 스타일(도트 밀도, 캐릭터 비율)과 엔진/해상도 결정이 먼저 필요하다. 이 결정 전에 구체적 목업(정확한 픽셀 크기, 타일 그리드 등)을 만들면 나중에 전부 다시 만들어야 할 위험이 크다.

**활성화 시 동작:**
1. 사용자에게 "그래픽 스타일/엔진이 아직 미확정 상태라 정식 목업 대신 러프 와이어프레임/구조만 만들 수 있다"고 먼저 알린다.
2. 정식 목업이 필요하면 `docs/format/game-format-tbd.html`와 `docs/art/mood-board.html`의 미확정 항목부터 결정하도록 안내한다 (open-question-facilitator 에이전트/스킬 활용 권장).
3. 그럼에도 러프 스케치가 필요하다면, 해상도/도트 크기에 무관한 **레이아웃 단위(그리드 칸 수, 상대적 비율)** 로만 와이어프레임을 만들고, 파일명에 `-rough` 접미사를 붙여 `docs/art/`에 저장한다. 정식 산출물처럼 취급하지 않는다.
4. 결정이 완료되면 이 스텁을 실제 목업 제작 절차(도구, 해상도 규격, 타일셋 규칙 포함)로 갱신해야 한다는 TODO를 스킬 파일 하단에 남긴다.
