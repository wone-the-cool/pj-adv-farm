# 미확정 항목 정리/논의 스킬

**목적:** 이 프로젝트는 "미확정 항목 임의 확정 금지, 옵션 제시" 원칙(CLAUDE.md)을 지키는 대신, 미확정 항목이 문서 곳곳에 흩어져 방치되기 쉽다. 이 스킬은 그것을 주기적으로 훑어 안건화하는 절차다.

**활성화:** "미확정 항목 정리해줘", "뭐가 안 정해졌지", "결정할 것 훑어줘", "다음에 뭐 논의하면 좋을지" 등.

**절차:**
1. `docs/risks.html`, `docs/**/*.html`에서 "미확정 — 상의 필요" 헤더가 있는 섹션을 Grep으로 모두 찾는다.
2. 안건을 파일 위치와 함께 짧은 목록으로 사용자에게 제시한다 (전부 한 번에 논의를 강요하지 않고, 개수와 주제만 먼저 보여준다).
3. 사용자가 특정 안건을 고르면:
   - 그 항목에 이미 옵션이 정리되어 있으면 그대로 제시.
   - 없으면 관련 에이전트(game-designer/technical-advisor/narrative-writer/systems-designer/scope-reviewer) 관점을 빌려 옵션+트레이드오프를 만든다.
   - 다수 안건을 깊게 순차적으로 다뤄야 하면 `open-question-facilitator` 에이전트로 넘겨 논의를 이어간다.
4. 사용자가 결론을 내리면 **반드시 그 자리에서 문서를 갱신**한다:
   - 해당 "미확정" 섹션에서 항목 제거/"확정" 표시
   - 관련 원칙/본문 섹션에 결정 사항 반영
   - `docs/risks.html` 상태 갱신
5. 절대 사용자 대신 결정하지 않는다 — 목록화와 옵션 제시, 그리고 결정 이후의 문서 반영까지만 수행한다.

**출력 형식 (1단계):**
```
현재 미확정 항목 (N개):
1. [파일 경로] 항목명 — 한줄 요약
2. ...
어느 것부터 논의할까?
```
